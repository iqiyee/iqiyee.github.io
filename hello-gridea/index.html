我到底为了什么写博客？

2023-03-14 Gridea

✍️ 建立博客初心就是写东西，本来只是想记录下我的生活、心情、知识、笔记、创意... ... 然而，却迷失在主题、功能、代码之中……

我到底为了什么写博客？

这个问题我已经想了好久了，我建博客是为了什么，似乎之前一直都是在折腾主题、优化、主机等……却忘记了博客的初心：写！

这个域名是2011年注册的，到现在12年了，可是到现在这个博客仍然没有超过五篇文章，有点讽刺。服务器3年前到期了，没有继续续费；所以域名备案去年阿里云给注销了，七牛云去年7月份也给停了。似乎我再也不会折腾了，但是我舍不得这个域名，虽然不咋地。

婚后有了孩子，重心也随之放在家庭和工作上了，每天过的生活可以称之为鸡飞狗跳的，在职场上与领导与客户“相互博弈”，在家与孩子“斗智斗勇”，时间它不允许我有过多的爱好。但我在夜深人静的时候就在想，我来这人间一趟，总得留点什么吧。亦或者等我老了，我可以翻看下现在写的东西，老年生活也不会那么枯燥了呢。

其实很可惜，那些年的记录丢的丢，损坏的损坏，基本没留下了。但既然没留下那就随它吧，此时此刻，我又重新上路开始了！

使用Gridea+𝖦𝗂𝗍𝗁𝗎𝖻 𝖯𝖺𝗀𝖾𝗌写博客

有一段时间，我用七牛搭建了纯html页面的博客，但是后来也不了了之，到现在，我看到了有人说利用github写博客，我的心情又激动起来了，但是在网上找了一堆教程都是Hexo 、Jekyll 等程序，来来回回折腾了几天，我都快放弃了，后来不死心，在bing上搜索“静态博客写作器”，竟然被我找到了宝贝，就是Gridea，下面是官方介绍：

📝 你可以使用最酷的 Markdown 语法，进行快速创作
🌉 你可以给文章配上精美的封面图和在文章任意位置插入图片
🏷️ 你可以对文章进行标签分组
📋 你可以自定义菜单，甚至可以创建外部链接菜单
💻 你可以在 Windows，MacOS 或 Linux 设备上使用此客户端
🌎 你可以使用 𝖦𝗂𝗍𝗁𝗎𝖻 𝖯𝖺𝗀𝖾𝗌 或 Coding Pages 向世界展示，未来将支持更多平台
🌁 你可以任意使用应用内默认主题或任意第三方主题，强大的主题自定义能力
官方下载链接：Gridea 主页

这不是就是我想要的嘛！😍不用考虑代码，不用关心服务器（托管在𝖦𝗂𝗍𝗁𝗎𝖻 𝖯𝖺𝗀𝖾𝗌），仅需一个域名即可。我尝试了部署了下，半个小时折腾好了（主要还是因为我没梯子，Github的速度你懂的😂）

第一次使用他需要你进行应用的初始化配置，才能让他和 Github Pages 连接。配置很简单，可以参考下面的方法进行配置：

域名：Github Pages 对应的域名（例如：https://iqiyee.github.io）
仓库名称：你的静态文件存放的仓库（例如：iqiyee.github.io）
分支：你的 Github Pages 对应的分支（例如：main）
用户名：仓库所属用户的用户名（通常就是你的 Github 账号的昵称）
邮箱：Git 推送时使用的邮箱（通常就是你的 Github 账号的邮箱）
Token: Git 推送时需要的 Token，用来向 Github 提交构建后的文件 🔗点此申请
CNAME: 可通过这个选项配置你自己的域名（例如：www.9doe.com）

至此博客就已经搭建好了，你可以设置其他的包括评论、主题等等内容了，自带主题真的很精致漂亮，完全不用去找其他主题了，专注于“写”吧。

emmm……我不该思考我为了什么写博客么，怎么变成了搭建博客了🤣
